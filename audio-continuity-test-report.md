### 1. 일반 링크 클릭 네비게이션
**목적**: 페이지 내 링크 클릭 시 오디오 연속성 확인

**테스트 단계**:
- [ ] 1. 홈페이지에서 음악 시작
- [ ] 2. 다른 페이지 링크 클릭
- [ ] 3. 오디오 상태 확인

**기대 결과**: 음악이 중단 없이 계속 재생됨
